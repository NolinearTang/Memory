## 界面功能说明

### 顶部状态栏

- **连接状态**：绿色圆点表示在线，红色表示离线
- **API地址**：可以修改后端服务地址
- **测试连接**：点击按钮测试API连接

### 左侧：添加对话消息

#### 步骤：

1. **输入Session ID**
   - 例如：`demo_session`, `session_001`
   - 同一个Session ID的对话会被归到一起

2. **输入元数据（可选）**
   - **产品信息**：如 `MD500, MD320`（逗号分隔）
   - **故障码**：如 `E001, E002`
   - **功能码**：如 `F100, F200`

3. **添加消息**
   - 点击 **User** 或 **Assistant** 选择角色
   - 输入消息内容
   - 点击 **➕ 添加消息**
   - 可以添加多条消息（一问一答）

4. **提交对话**
   - 点击 **🚀 提交对话**
   - 后台会自动处理：
     - 保存所有消息
     - 生成摘要（3轮以上）
     - 压缩长回答（>200字）

### 右侧：获取对话上下文

#### 步骤：

1. **输入Session ID**
   - 与左侧相同的Session ID

2. **输入查询关键词**
   - 例如：`故障处理`, `散热问题`, `轴承`
   - 系统会用BM25检索相关对话

3. **查询上下文**
   - 点击 **🔍 查询上下文**
   - 会显示：
     - 📊 状态信息（产品、故障码、功能码）
     - 💬 上下文信息（摘要、近期对话、相关对话）

4. **查看所有会话**
   - 点击 **📋 查看所有会话**
   - 显示所有Session ID
   - 点击Session ID可快速加载

## 使用示例

### 示例1：模拟故障排查对话

1. **添加第一轮对话**
   ```
   Session ID: fault_debug_001
   产品信息: MD500
   故障码: E001
   
   User: 变频器显示E001错误，怎么处理？
   Assistant: E001是过热保护报警。请检查：1）环境温度是否过高 2）负载是否过大...
   ```

2. **添加第二轮对话**
   ```
   继续使用相同Session ID
   
   User: 负载正常，环境温度也不高
   Assistant: 那可能是散热系统问题。请检查散热风扇是否正常运转...
   ```

3. **添加第三轮对话**
   ```
   User: 风扇转速很慢
   Assistant: 风扇转速慢是过热的主要原因。请更换散热风扇...
   ```

4. **查询上下文**
   ```
   Session ID: fault_debug_001
   查询: 风扇
   
   系统会返回：
   - 对话摘要（自动生成）
   - 近3轮对话
   - 与"风扇"相关的对话片段
   ```

### 示例2：测试自动摘要

2. **查询上下文**
   - 输入任意关键词
   - 查看"对话摘要"部分
   - 前面的对话已被压缩成摘要

### 示例3：测试BM25检索

1. **添加多个主题的对话**
   ```
   话题1: 过热问题
   话题2: 通讯故障
   话题3: 参数设置
   ```

2. **使用关键词检索**
   ```
   查询: 过热

   查询: 通讯
   → 返回通讯相关的对话
   ```

## 常见问题

### Q: 页面打不开？
**A:** 
- 确认服务器已启动
- 检查端口8789是否被占用
- 尝试访问 http://127.0.0.1:8789/

### Q: 提交对话后无响应？
**A:**
- 打开开发者工具查看错误
- 检查API地址是否正确
- 查看服务器日志

### Q: 摘要未生成？
**A:**
- 等待几秒后再查询
- 确认对话轮数超过3轮
- 检查LLM API配置

### Q: CORS错误？
**A:**
- 确认后端已添加CORS中间件
- 检查浏览器控制台的具体错误
- 尝试刷新页面

